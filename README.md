# DTMF Encoder / Decoder (BIL216 – İşaretler ve Sistemler)

Bu proje, Türk alfabesindeki 29 harf ve boşluk karakteri için DTMF benzeri iki frekanslı sinyaller kullanarak:

- Metinden sese dönüşüm (Encoding)
- Sesten metne çözümleme (Decoding)

işlemlerini gerçekleştiren bir Python uygulamasıdır.

## Özellikler
✔ 30 karakterlik Türkçe DTMF frekans tablosu  
✔ 40 ms karakter süreli ton üretimi  
✔ FFT tabanlı frekans çözümleme  
✔ Gürültü toleranslı çözümleme algoritması  
✔ Modern arayüz (CustomTkinter)  
✔ JSON tabanlı frekans eşlemesi
